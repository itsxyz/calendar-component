# 📅 Interactive Wall Calendar

A polished, interactive calendar component built with **Next.js** and **Tailwind CSS**.

## ✨ Features
- Wall calendar aesthetic with a hero image per month
- Mood-based color theme that changes every month
- Date range selection with clear visual states
- Notes section tied to selected date range (saved with localStorage)
- Month navigation with smooth animations
- Holiday markers for Indian public holidays
- Fully responsive — works on mobile and desktop

## 🛠 Built With
- Next.js 16
- Tailwind CSS
- localStorage for data persistence

## 🚀 Run Locally
npm install
npm run dev

Open http://localhost:3000 in your browser.
